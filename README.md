## 包管理器使用 [yarn](https://yarnpkg.com/getting-started/install)   

### 安裝依賴
```
yarn
```

### dev 模式測試網頁
```
yarn dev
```

### 編譯網頁
```
yarn build
```

### 測試編譯後的網頁
```
yarn serve
```

### 清除編譯內容
```
yarn clear
```

### 部署到 github page
```
yarn deploy
```
