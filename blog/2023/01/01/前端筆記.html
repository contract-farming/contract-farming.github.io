前端筆記

January 1, 2023 · 2 min read

編寫環境​

前端框架 vue.js (使用 vue 3 + vite)
前端介面 Bootstrap 5
使用工具 vs code

參考教學​

  • vue.js 官方文檔
  • bilibili教學
  • 重新認識 Vue.js
  • iT邦幫忙

Vite

初始化​

# 使用 npm 安裝
npm init vue@latest

執行 server​

# step 1 先 install node_modules
npm i

# step 2 執行 server
npm run dev

Bootstrap 5

bootstrap 5 安裝​

# 使用 npm 安裝
npm install bootstrap
TIP​

使用 bootstrap 建議一併引入 Sass
{: .block-tip }

# 使用 npm 安裝
npm install -D sass

在專案中使用 bootstrap​

在 main.js 中引入

// 注意先後順序
import "bootstrap/dist/css/bootstrap.css";
import "bootstrap/dist/js/bootstrap.js";

在要使用 bootstrap 樣式的地方去官網 (ctrl + c) + (ctrl + v) :D

<!-- 例如 引入一個導航欄 -->
<nav class="navbar bg-body-tertiary">
  <div class="container-fluid">
    <a class="navbar-brand" href="#">
      <img src="/docs/5.3/assets/brand/bootstrap-logo.svg" alt="Logo" width="30" height="24" class="d-inline-block align-text-top">
      Bootstrap
    </a>
  </div>
</nav>
